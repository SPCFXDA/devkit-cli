Devkit CLI Manual

The devkit-cli command-line interface provides a set of commands for managing a development node and interacting with various components. Below are the available commands and their usage.

start

Start the development node.

Options:
  • --logs - Display logs after starting (default: false)
  • --scan - Scan the node for transactions after starting (default: false)

Example:

devkit-cli start --logs

stop

Stop the development node.

Example:

devkit-cli stop --force

scan

Scan the node for transactions.

Example:

devkit-cli scan

status

Show the node status.

Example:

devkit-cli status

logs

Show node logs.

Example:

devkit-cli logs

errors

Show any errors the node produced.

Example:

devkit-cli errors

balance [address]

Show the balance of an address. If no address is specified, show the genesis account's balance.

Options:
  • address - Wallet destination address

Example:

devkit-cli balance 0x1234567890abcdef

faucet [amount] [address]

Faucet utility to fund Core and ESpace wallets.

Options:
  • amount - Amount of CFX to send
  • address - Wallet destination address

Example:

devkit-cli faucet 10 0x1234567890abcdef

wallet

Configure a local HDWallet.

Example:

devkit-cli wallet

*

Catch-all for unknown commands. Prints an error message.

Example:

devkit-cli unknownCommand

For more details, use devkit-cli -h to display the help information.
